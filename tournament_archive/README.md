# Tournament Archive Shortcode

This snippet creates an automatic archive overview for recurring club tournaments in WordPress.

## The problem

Many clubs create a new page for every annual tournament:

* Blitz Championship 2024
* Blitz Championship 2025
* Blitz Championship 2026
* City Championship 2024
* City Championship 2025
* ...

These pages contain valuable historical information. However, maintaining archive pages manually becomes time-consuming and error-prone.

## The solution

This snippet automatically creates an archive overview by collecting tournament pages based on WordPress post meta values.

It:

* finds all tournament pages with matching year metadata
* sorts them by year (newest first)
* highlights the current tournament
* lists previous tournaments automatically

When a new tournament page is created, the archive updates without any manual changes.

## Requirements

This snippet relies on metadata created by our own WordPress plugins:

* [monatsblitz-wp-plugin](https://github.com/kindermaenner/monatsblitz-wp-plugin)
  Creates tournament pages and stores the corresponding metadata for monthly blitz tournaments.

* [swisschess-wp-plugin](https://github.com/kindermaenner/swisschess-wp-plugin)
  Creates tournament pages for Swiss-system tournaments and provides the metadata used by the archive.

The archive snippet does not create tournament data itself. It reads the metadata generated by these plugins and turns it into a user-friendly overview.

## Usage

Add the shortcode:

```text
[turnier_archiv]
```

to the page where the archive should appear.

## Customization

The tournament types are configured in the snippet itself. Add or remove entries depending on the tournament categories used on your own website.

Example:

```php
$turniere = [
    'blitzmeisterschaft' => 'Blitzmeisterschaft',
    'stadtmeisterschaft' => 'Stadtmeisterschaft'
];
```

## Background

This solution was developed for a small chess club website. The goal was to keep current tournaments easy to find while preserving access to the complete tournament history without manual maintenance.
