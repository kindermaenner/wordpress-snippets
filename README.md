# wordpress-snippets

This repository contains the small but essential building blocks that help us run and maintain our non-profit club websites:
– custom PHP snippets (managed via the Code Snippets plugin)
– lightweight custom CSS for layout fixes
– practical WordPress enhancements that solve real-world problems

We maintain two WordPress websites for local clubs, both built and operated with minimal budgets and without commercial plugins. These snippets document how we approach common challenges in a clean, reusable and transparent way.

The focus is not on replacing established plugins, but on filling the gaps where a few lines of custom code provide the best solution.

Feel free to explore, reuse, and adapt anything that helps improve your own WordPress setup.

# Disclaimer

These snippets are provided as examples from our own projects. Please test changes in your own environment before using them on a production website.
